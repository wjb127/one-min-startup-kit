# 1분 창업 키트

AI를 활용한 랜딩 페이지 생성 및 MVP 테스트 서비스

## 주요 기능

- AI 랜딩 페이지 생성
- MVP 테스트 (가상 결제 버튼, 이메일 수집)
- 실시간 데이터 분석
- 구글 애즈 연동

## 기술 스택

- Next.js
- TypeScript
- Tailwind CSS
- Supabase

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```
